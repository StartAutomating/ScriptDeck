File Path

The path to the file or script to run.

Working Directory

The Working Directory

Argument List Script

The result of this script will be passed to the -ArgumentList parameter of Start-Process.

If no script is provided, the process will be started without arguments

Verb

The Verb used to launch the process.

Versb are used to perform alternate actions on a given proccess.

As examples: the verb RunAs will attempt to launch a process as administrator, and the verb Print will attempt to print a document.

When Loaded

Runs whenever the button is loaded.

This is where any PowerShell modules should be imported, and any state should be set up (for instance, discovering devices)

When Unloaded

Runs whenever the button is loaded.

This is where any state should be cleaned up. Unloading modules is not recommended due to the potential of another button using the same module.
